# ATV01_PAM_POO

Repositório referente a entrega da atividade Lista de exercicios, da materia Programação Mobile, do professor Jacques Simão

# Como foi feito?

- Estudo de Kotlin: Assisti muitas video-aulas no YouTube.
- Foco no Básico: Aprendi a sintaxe de estruturas essenciais, como o que são e como usar Maps (mapas/dicionários).
- Organização do Código: Entendi a estrutura do Kotlin, usando fun main() para o início e funções separadas (fun suaFuncao()) para modular o código.

O objetivo foi entender a lógica e a estrutura básica para resolver a lista.
